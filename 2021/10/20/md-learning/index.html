GitHub Flavored Markdown Spec

一、 文档分块

  • 以0-3个空格开头且后跟多于三个-,_,*字符的形式分割主题块

    建议以---,___,***为一行分割主题

  • markdown同HTML分为六级主题，以1-6个#字符，分别表示1-6级主题

    # 一级主题
    ## 二级主题
    ### 三级主题
    #### 四级主题
    ##### 五级主题
    ###### 六级主题
